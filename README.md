# Portfolio v2 (2021)
## Description
My portfolio by Thomas Bichot.   
This project use Nuxt.js and TailwindCSS.

## Installation
```
$ npm install nuxt
```
Then install the various components required like TailwindCSS.

## Usage 
### Local (dev)
```
$ npm run dev
```

### Server
```
$ npm run build
$ pm2 start -- start
```
